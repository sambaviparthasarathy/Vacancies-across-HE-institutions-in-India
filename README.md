# Vacancies-across-HE-institutions-in-India
This file contains RTI data set regarding staff vacancies across Centrally funded institutions
While RTI replies have been uploaded as received, compiled data can be found in the file named: Vacant posts data - compiled data.csv
To understand data availability, please check: Institution wise data availability.csv
Data for the charts were obtained through requests under the Right to Information Act. The responses were received between January and March, 2026
